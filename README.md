# Derail
Completely destroys a rails project: Databases, Chewy indices (if any) and deletes the project directory.

## Usage
```bash
rake derail:project
```

## Installation
Add this line to your application's Gemfile:

```ruby
group :development do
  gem 'derail'
end
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install derail
```
